■このゲームについて
・ゲームタイトル　ワ釣り
・ジャンル　　　　釣り
・難易度　　　　　かんたん
・プレイ時間　　　2分～

このゲームは「わ」をテーマにしたゲームジャムで作成されたものです。
"ワ"で魚(?)を釣り、高得点を目指しましょう。

■起動方法
ダウンロードしたファイルを解凍して全て同じフォルダに置き、「wa_fishing.exe」を実行してください。

■基本操作
WASD or 方向キー：移動
escキー:ゲームの終了
